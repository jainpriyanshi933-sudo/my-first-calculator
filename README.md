# my-first-calculator
This is a simple calculator web application built using HTML, CSS, and JavaScript. It performs basic arithmetic operations like addition, subtraction, multiplication, and division. It also includes a clear function to reset the input.

🧮 Simple Calculator

A fully functional web-based calculator built using HTML, CSS, and JavaScript.

This project performs basic arithmetic operations and demonstrates fundamental front-end development skills such as DOM manipulation, event handling, and responsive design.

🚀 Features
➕ Addition
➖ Subtraction
✖ Multiplication
➗ Division
🧹 Clear button to reset all values
⚡ Instant result using "=" button
🛠 Technologies Used

HTML – Structure of the calculator
CSS – Styling and layout design
JavaScript – Logic and functionality
